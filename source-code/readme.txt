Hướng dẫn chạy
1. Vô Google Drive, tạo thư mục 'colab/demand-forecast' và up file 'gld.csv' trong thư mục 'data' lên
2. Vô Google Colab https://colab.research.google.com/ up file 'code.ipynb' lên Google Colab 
3. Chọn Runtime -> Run all để chạy. Lưu ý, trong lúc chạy sẽ thông báo truy cập vào link Google Drive để cấp quyền truy cập vào Google Drive ở cell thứ 2.
